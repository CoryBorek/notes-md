# The Problem of Sorting

## Input
Sequence $<a_1, a_2, ..., a_n>$ of numbers.

## Output
Permutaition $<a'_1, a'_2, ..., a'_n>$ such that $a'_1 < a'_2 <  ... < a'_n$.

### Example

 - Input : 8 2 4 9 3 6
 - Output: 2 3 4 6 8 9
